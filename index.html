LUNA — TurboPi Intelligent Robot

QUT University Project · Jul – Nov 2025 · Python · Raspberry Pi · Computer Vision

What it is

A Raspberry Pi 4B-based intelligent robot built on the Hiwonder TurboPi platform. The system integrates hardware assembly, Python programming, wireless control, computer vision, offline voice interaction, and PID-tuned movement into one cohesive robotics platform — built collaboratively with a partner from the ground up.

Why I built it

This started as a university robotics unit but became something I actually cared about finishing. I wanted to understand how software decisions translate into real physical behaviour — how a bad control loop causes a robot to spin, or how a missed sensor reading ends a demo early. You can't ignore that kind of feedback. It keeps you honest.

"This was the first project where I could touch what I'd built. Every bug had a physical consequence. You could hear when something wasn't working."
What I explored

How do you integrate independent hardware subsystems — vision, voice, movement, web control — into one robot that responds reliably under real conditions? The core challenge wasn't building individual features. It was making them coexist without breaking each other, and understanding the failure modes when they do.

Tech stack
PythonFlaskOpenCV Raspberry Pi 4BWonderEchoI²C NumPyTkinter
What I learned
  • Hardware debugging is different from software debugging — consequences are immediate and visible
  • Defensive programming matters most in embedded systems where failure has physical impact
  • System integration is its own discipline, not just "connecting parts" together
  • Documenting decisions in real time saves the project when things break at 2am
